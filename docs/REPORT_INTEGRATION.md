# 🔄 Airport Web-Desktop 統合自動化 報告書

本プロジェクトの集大成として、Webブラウザとデスクトップアプリケーションを横断する統合ワークフローの自動化実験を行いました。

## 🎯 ミッション概要
**「Webで東京の天気を調べ、その結果をデスクトップのメモ帳アプリに記録し、ファイルとして保存する」**

このタスクは、以下の複合的な操作能力を必要とします。
1.  **Web操作**: ブラウザ起動、検索エンジン利用。
2.  **視覚的情報抽出**: 検索結果画面からの非定型データの読み取り (Vision)。
3.  **情報保持**: 読み取ったデータのメモリ内保持。
4.  **デスクトップ操作**: OSネイティブアプリ（テキストエディタ）の起動。
5.  **データ転記**: 保持した情報の入力。
6.  **ファイル操作**: アプリの保存ダイアログの制御。

## 🛠️ 技術的ブレイクスルー

1.  **クロスコンテキスト変数**:
    Web操作フェーズ (`src.main.ATC`) で取得した情報を変数に格納し、デスクトップ操作フェーズ (`src.desktop_controller.DesktopATC`) に引き渡すパイプラインを確立しました。

2.  **OSシステムダイアログの制御**:
    Vision操作（保存メニューのクリック）とキーボードショートカット（Ctrl+S）、そしてOSのファイル保存ダイアログへのパス入力（`pyautogui.write`）を組み合わせ、確実なファイル保存を実現しました。
